    DHCP Server using FreeBSD

    DHCP is a protocol which use by computers in the network to determine parameters like default gateway, subnet mask, IP address and DNS from a DHCP server. The general purpose of a DHCP server is to make IP address allocation easier and IP setting more dynamic. It help computer user so not need to set IP when there is an addition of new user or after reformat a computer.
    More information about DHCP can be read on:

    http://en.wikipedia.org/wiki/Dynamic_Host_Configuration_Protocol#Introduction
